## The Project

A demo e-commerce application for LEGO products, built as a learning project to explore modern e-commerce patterns and deployment workflows.

## My Role

**Developer** — Built the complete application:

- **Frontend Development** — Product catalog and cart functionality
- **UI/UX** — Clean shopping experience for collectibles
- **Deployment** — Vercel hosting and CI/CD setup

## Technical Stack

| Layer | Technology |
|-------|------------|
| Framework | Next.js |
| Hosting | Vercel |
| Styling | Modern CSS |

## Purpose

This project served as a hands-on exploration of:

1. **E-commerce Fundamentals** — Product listings, cart state, checkout flows
2. **Next.js Patterns** — Server components, routing, data fetching
3. **Vercel Deployment** — Preview deployments, edge functions

## What I Learned

- **E-commerce state management** — Cart persistence and synchronization challenges
- **Product catalog design** — Filtering, sorting, and search UX patterns
- **Deployment workflows** — Vercel's preview deployments for rapid iteration
