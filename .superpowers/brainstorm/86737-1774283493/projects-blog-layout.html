Design Notes

  • /projects: Status filter pills (All/Live/Experiment/Archived) + 2-col card grid. Same card style as Home featured.
  • /blog: Search input + list rows with title, date (monospace), description, and tag pills. No cards — clean list.
  • /projects/[slug]: Back link, title + status badge, tagline, tech stack pills, live/source links, then MDX body.
  • /blog/[slug]: Back link, date + reading time (monospace), title, tags, inline TOC card, then MDX body. Share links at bottom.
  • Both detail pages: Max-width ~640px for comfortable reading. Back link always visible at top.
